Welcome to xTherm

xTherm is an open source CPU temperature monitor for mac OS X. xTherm displays as a status bar menu showing CPU temperature and fan speed. Logs of temperature are kept for two days.

Supported Systems

OS X 10.11 (El Capitan)

Installing

  • Download the zip file containing a compiled binary
  • Right-Click on the binary and click 'open'
  • (optional) Add to your startup items
